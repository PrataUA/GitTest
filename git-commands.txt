Для ініціалізації репозиторія - git init . 
Щоб зрозуміти існуючий стан репозиторія - git status
Щоб додати файл в staging area - git add <ім'я файлу>
Щоб зробити комміт - git commit -m <"Опис комміта">
//Щоб подивитись історію коммітов - git log
Для переходу до конкретного комміту - git checkout <вказуємо hash комміту>
